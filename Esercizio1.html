Domanda 1: A cosa serve Bootstrap?

Bootstrap è un framework front-end open-source utilizzato per la creazione di siti web responsive e stilizzati.

Domanda 2: Cosa devi importare per creare una pagina web con Bootstrap?

Per creare una pagina web con Bootstrap, devi importare il CSS e il JavaScript di Bootstrap. Ecco un esempio di come farlo:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/ css/bootstrap.min.css">

Domanda 3: Cos'è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server distribuiti in tutto il mondo che fornisce contenuti, come file CSS e JavaScript, agli utenti da server fisicamente vicini a loro. Utilizzare un CDN per Bootstrap offre diversi vantaggi rispetto a scaricare Bootstrap e ospitarlo sul tuo sito:

  • Velocità: I file vengono caricati più rapidamente poiché vengono forniti da server CDN ottimizzati e distribuiti geograficamente.
  • Affidabilità: I server CDN sono progettati per gestire carichi di lavoro elevati e possono migliorare l'affidabilità del tuo sito.
  • Risparmio di larghezza di banda: Riduci il carico del tuo server poiché i file vengono serviti da un CDN.

Domanda 4: A cosa serve l'attributo "content" del tag meta con il valore "width=device-width, initial-scale=1"?

L'attributo "content" del tag meta con il valore "width=device-width, initial-scale=1" è utilizzato per rendere una pagina web responsive su dispositivi mobili. Questa impostazione imposta la larghezza del viewport (la finestra visualizzata sul dispositivo) sulla larghezza del dispositivo stesso e imposta il livello di zoom iniziale a 1. Ciò significa che la pagina verrà visualizzata correttamente su dispositivi di diverse dimensioni, adattando la larghezza e il livello di zoom iniziale per adattarsi al dispositivo specifico.

Domanda 5: Che differenza c'è tra Container e Container Fluid?

La differenza principale tra "Container" e "Container Fluid" in Bootstrap riguarda la larghezza e la fluidità del contenitore:

  • "Container": Il contenitore standard "container" ha una larghezza fissa e si adatta al centro della pagina. È adatto per la creazione di layout con larghezza fissa.
  • "Container Fluid": Il contenitore "container-fluid" è al 100% della larghezza del viewport e si estende per tutta la larghezza del dispositivo. È ideale per creare layout fluidi e completamente responsivi.
Puoi scegliere tra i due in base alle esigenze del tuo layout.
